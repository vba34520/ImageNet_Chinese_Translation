# ImageNet_Chinese_Translation

对 [ImageNet千分类标签](https://storage.googleapis.com/download.tensorflow.org/data/imagenet_class_index.json) 进行翻译

# 部分翻译

```text
tench 鲤鱼
goldfish 金鱼
great_white_shark 大白鲨
tiger_shark 虎鲨
hammerhead 锤头鲨
electric_ray 电鳐
stingray 黄貂鱼
cock 公鸡
hen 母鸡
ostrich 鸵鸟
brambling 燕雀
goldfinch 金翅雀
house_finch 家雀
junco 灯芯草雀
indigo_bunting 蓝鹀
```
